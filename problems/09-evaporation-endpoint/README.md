# #09 — The endpoint of evaporation: remnants vs information release

*Also known as: black hole remnants, final state of evaporation, Planck-scale remnant problem.*

> **Status: OPEN** · Statement tier: `established_theory` · Resolution tier: `open_conjecture` · First posed: 1974-1992.

## The problem

As a black hole evaporates via Hawking radiation it shrinks toward the Planck scale, where the semiclassical description fails. What is the final state? Candidates include: a stable or long-lived Planck-mass remnant retaining the lost information; complete evaporation with information carried out in subtle correlations of the radiation; information stored in a disconnected 'baby universe'; or a genuine breakdown of unitary evolution. Each option has serious difficulties (e.g. remnants must carry unboundedly large entropy for a given mass, threatening infinite pair-production), and none is established.

- **Stephen Hawking** — 1974-76 evaporation and the information question that frames the endpoint.
- **Steven Giddings** — 1992 analysis of massive remnants and their problems.

## Key ideas

- Semiclassical gravity breaks down once the hole reaches the Planck mass (~10^-8 kg), so the true endpoint needs quantum gravity.
- Stable remnant: a Planck-scale object stores the information — but must have enormous degeneracy at fixed mass, risking divergent pair-production (the 'remnant problem').
- Complete evaporation + information in correlations: requires the late radiation to be highly non-thermal in a way the leading calculation misses (now supported by the island/Page-curve results, #02).
- Baby universe: information escapes into a causally-disconnected region (does not help an outside observer recover it).
- Long-lived (metastable) remnant as a phenomenological middle ground.
- The answer is constrained by, and feeds back into, the information paradox and its island resolution.

## Live approaches

- **Remnant scenarios** (`speculative_extension`) — Stable or long-lived Planck-scale objects retaining information; analyze consistency (pair-production, entropy bounds). _Proponents: Steven Giddings, Yakir Aharonov, Tom Banks._
- **Complete evaporation with unitary radiation** (`open_conjecture`) — Information is in the full radiation state; supported by Page-curve / island computations. _Proponents: Don Page, Ahmed Almheiri, Geoffrey Penington._
- **Baby universes / disconnected interiors** (`speculative_extension`) — Information passes into a disconnected region; assessed against the gravitational path integral. _Proponents: Stephen Hawking, Tom Banks._
